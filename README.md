# 如何使用：
1. 請先下載exe及xml檔案
2. 打開xml檔案
3. 輸入正確股票代碼至stockNo的欄位內
4. 輸入欲查詢的時間範圍，共有4個：startYear, startMonth, endYear, endMonth
5. 記得在excelName裡輸入欲輸出的excel名稱(不須輸入副檔名)
6. 輸入完記得存檔關閉
7. 執行exe
8. 最後即可獲得一個整理好的excel報表
